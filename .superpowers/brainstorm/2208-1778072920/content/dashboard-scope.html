Trust Dashboard Scope

The dashboard will be the primary interface for the quarantine queue. What else should this dashboard encompass?

A

Memory & Quarantine Focus Only

Focus strictly on pending memory candidates, source evidence, contradiction alerts, and confidence scores. Keeps it lightweight.

B

Broad Autonomy Audit (The "Control Room")

Includes the memory queue, plus recent autonomous actions, API spend, tool usage history, and pending execution approvals.

C

System Health & Knowledge

Includes the memory queue, plus a high-level view of the knowledge map, mood timeline, and database health metrics.
