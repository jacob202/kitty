01 — what changed

we kept the soul and threw out the skin

was
  • warm-CRT terminal, 8-bit everything
  • orange-on-dark, forever
  • mono + pixel type, generic now
  • every screen, the same rhythm
now
  • bone-bright by day, deep-indigo by night
  • electric cobalt runs the show
  • type with a spine — bricolage + hanken
  • tactile, weighty, a thing you touch
kept
  • the four states — idle/working/done/broke
  • the deadpan, lowercase voice
  • the cat is an instrument, not a portrait
  • built for an audience of one
