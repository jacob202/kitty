Think of an embedding as a coordinate for meaning. You take text and run it through a model that produces a fixed-size float array — say, 1536 dimensions for text-embedding-3-small.

Two sentences that mean similar things produce vectors that are geometrically close in that space. That proximity is measurable with cosine similarity — the dot product of the unit vectors.
# cosine similarity between two embeddings import numpy as np def cosine_sim(a, b): return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b)) sim = cosine_sim(embed("cat"), embed("feline")) # → ~0.91 sim = cosine_sim(embed("cat"), embed("database")) # → ~0.21
