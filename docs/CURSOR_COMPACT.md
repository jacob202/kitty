# Cursor: Compact chat — Kitty shortcut

Cursor has a dedicated **Compact chat** action that trims transcript **without deleting repo files**.

**Do this every time before compact:**

1. Update **`SESSION_HANDOFF.md`** (root) with anything the next session must not re-infer.
2. Optional: glance at **`AGENTS.md`** if you changed rules or stack.

**After compact:** start the next chat with **`@SESSION_HANDOFF.md`** (or “read SESSION_HANDOFF first”).

The **same habit** across all other AI tools (Claude, Gemini, Codex, …) is documented in **`docs/SESSION_COMPACT_WORKFLOW.md`** — read that for the universal pattern.
