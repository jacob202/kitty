*Weeks 1-2. Ship a reactive but personality-rich Kitty with voice and ambient awareness.*

## Phase 2: Agent System & Background Tasks (free-code port)

*Weeks 3-4. Give Kitty the ability to spawn autonomous agents and run background tasks.*

**Problem:** Kitty has no way to spawn workers for parallel or background work. Everything is synchronous request-response.
